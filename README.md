### Описание проекта
Угадать загаданное компьютером число за минимальное количество попыток.

### Какой кейс решаем?
Нужно написать программу, которая угадывает число за минимальное количество попыток

:arrow-_up [к оглавлению]()

**Условия соревнования:**
- компьютер загадыает целое число от 0 до 100, нам нужно его угадать. Под "угадать" подразумевается написать программу, которая угадывает число.
- Алгоритм учитывает информацию о том, больше или меньше случайное число по отношению к нужному нам числу

**Метрика качества**
Результаты оцениваются по среднему количеству попыток при 1000 повторений

**Что практикуем?**
Учимся писать хороший код на Python
